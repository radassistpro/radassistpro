# RadAssistPro Website

Premium marketing website for [RadAssistPro](https://radassistpro.com) — Virtual PACS Admins and Prelim Teleradiologists for U.S. radiology.

## Pages

- **Home** — Video hero, stats, services preview, testimonials, contact form
- **About** — Mission, values, team approach
- **Services** — Virtual PACS Admins and Prelim Teleradiologists
- **How It Works** — Enquiry to onboarding roadmap

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS 4
- Framer Motion
- Lucide Icons
